# mean_web_survey_service

MEAN Stack 기반 설문조사 웹 어플리케이션

- 테스트 주소 : https://still-hollows-5516.herokuapp.com/


## 프로그램의 기능

- 로그인 
- 회원가입 
- 로그인 이후 설문을 수행하기
- 회원정보 수정하기
- 회원정보 삭제하기
- 회원정보 조회하기
- 설문 등록하기
- 설문 삭제하기
- 설문 수집하기
- 설문 결과보기
